# TODO List

A simple and interactive TODO List application to help you manage your daily tasks efficiently. [page:1]

## Features

1. **Create tasks**  
   You can add new tasks using the input box at the top of the app. [page:1]

   <img width="2560" height="1600" alt="TODO List overview" src="https://github.com/user-attachments/assets/1f8048d7-1ce6-4810-a439-f078322b6077" /> [page:1]

2. **Enter your task**  
   Type your task in the input field to add it to the list. [page:1]

   <img width="554" height="129" alt="Task input" src="https://github.com/user-attachments/assets/a5017897-4c6f-4560-8b73-f25ccbacd302" /> [page:1]

3. **Add tasks to the list**  
   Click on the “Add” button to save your task in the tasks list. [page:1]

   <img width="350" height="72" alt="Add button" src="https://github.com/user-attachments/assets/7c29b5a1-58e7-4fce-afc9-ee3c509580c2" /> [page:1]

4. **View all tasks**  
   All your tasks are displayed in a list so you can see everything you need to do at a glance. [page:1]

   <img width="733" height="906" alt="Tasks list" src="https://github.com/user-attachments/assets/1bb90715-2aa5-48a9-b968-7b403bc52a38" /> [page:1]

5. **Remaining tasks count**  
   The app shows the number of remaining (incomplete) tasks. [page:1]

   <img width="286" height="53" alt="Remaining tasks count" src="https://github.com/user-attachments/assets/d5161028-09bd-4572-8616-f433f87edc4b" /> [page:1]

6. **Filter pending tasks**  
   You can view only the tasks that are still pending. [page:1]

   <img width="656" height="566" alt="Pending tasks" src="https://github.com/user-attachments/assets/76445170-4a37-4074-b185-1c90463cd599" /> [page:1]

7. **Filter completed tasks**  
   You can also view only the tasks that you have completed. [page:1]

   <img width="715" height="372" alt="Completed tasks" src="https://github.com/user-attachments/assets/0a685e9b-8c79-4bfb-962c-49c48ec210c8" /> [page:1]

8. **Edit tasks**  
   If you make a mistake or your task changes, you can edit an existing task. [page:1]

   <img width="644" height="134" alt="Edit task" src="https://github.com/user-attachments/assets/ef7f293f-3fe1-4633-ad31-4f08003ee2a1" /> [page:1]

9. **Delete tasks**  
   You can delete tasks that you no longer need from the list. [page:1]
